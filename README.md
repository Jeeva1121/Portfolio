## 🚀 Features

- **Dynamic Hero Section:** Engaging 3D-style developer ID card with floating elements and a sleek introduction.
- **Premium Animations:** Buttery-smooth scroll animations, micro-interactions, and a custom magnetic cursor powered by **Framer Motion**.
- **Modern UI/UX:** Combining Glassmorphism and soft Neobrutalism aesthetics with a clean typography hierarchy (Outfit & Poppins).
- **Projects Showcase:** An interactive grid displaying top projects with hover states and fast image loading.
- **Certifications Carousel:** A touch-friendly sliding indicator carousel showcasing professional certificates with official logos.
- **Working Contact Form:** Fully functional contact form integrated with **Web3Forms** (No backend required!).
- **Fully Responsive:** Tailored experiences for both desktop (hover effects, 3D tilt) and mobile (performance-optimized touch scroll).

## 🛠️ Tech Stack

- **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Iconify](https://iconify.design/)

## 💻 Getting Started

### Prerequisites
Make sure you have Node.js (v18 or higher) and npm installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Jeeva1121/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to view the portfolio!

## ✉️ Contact Form Setup (Web3Forms)

This portfolio uses Web3Forms to handle contact form submissions for free without a backend. To set it up for yourself:
1. Go to [Web3Forms](https://web3forms.com/).
2. Enter your email to get a free **Access Key**.
3. Open `src/components/Contact.tsx`.
4. Replace the existing `access_key` in the `handleSubmit` function with your new key.
